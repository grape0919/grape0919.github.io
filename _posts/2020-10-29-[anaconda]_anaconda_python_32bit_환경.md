# Anaconda 32bit

64bit OS에서 아나콘다를 사용할 때,
32bit 가상환경을 만들어야 하는 상황에서
인터넷에 검색에서 바로 나오는

```sh
set CONDA_FORCE_32BIT=1
conda create -n py37_32
```

만 갖고는 제대로된 32bit 환경이 만들어 지지 않음...

### git log to excel

```sh
git log --all --date=format:'%Y-%m-%d %H:%M:%S' --pretty=format:%an,%ad,%s > ./history.csv
```

<!--more-->

---
| 옵션 | 설명 |
|:---:|:---:|
|%H| 커밋 해시|
|%h|짧은 길이 커밋 해시|
|%T|트리 해시|
|%t|짧은 길이 트리 해시|
|%P|부모 해시|
|%p|짧은 길이 부모 해시|
|%an|저자 이름|
|%ae|저자 메일|
|%ad|저자 시각 (형식은 –-date=옵션 참고)|
|%ar|저자 상대적 시각|
|%cn|커미터 이름|
|%ce|커미터 메일|
|%cd|커미터 시각|
|%cr|커미터 상대적 시각|
|%s|요약|

--pretty=format:  옵션으로 출력되는 로그 형태를 자유 자제로 수정 할 수 있다.

 
ex) ```--pretty=format:'log-',%an,%ad,%s,'end-log'```
